## 電話番号

```javascript
let reg = /^\d{2,4}-*\d{2,4}-*\d{2,4}$/;

console.log(reg.test("00-1111-2222"));
// >> true
console.log(reg.test("000-1111-2222"));
// >> true
console.log(reg.test("0000-11111-2222"));
// >> false
```

## メールアドレス

```javascript
let reg = /^[^\s@]+@[^\s@]+\.[^\s@]{2,}$/i;

console.log(reg.test("abc@example.com"));
// >> true
console.log(reg.test("abc123@example.co.jp"));
// >> true
console.log(reg.test("abc@com"));
// >> false
```
